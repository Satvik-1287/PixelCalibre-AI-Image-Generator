# PixelCalibre-AI-Image-Generator
This is an AI Image Generator created using HTML, CSS and JavaScript, and Pollinations.ai API. It creates enough good images for normal use. It is a basic project understanding the fundamentals of AI Images using Pollinations.ai free API based on URL Schemes and unique seeds for every image.
